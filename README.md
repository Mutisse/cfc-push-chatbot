# 🏛️ CFC PUSH Chatbot

Chatbot WhatsApp para a Igreja CFC PUSH - Oramos Até Algo Acontecer!

## 🚀 Funcionalidades

- ✅ Menu interativo via WhatsApp
- ✅ Cadastro de novos membros  
- ✅ Pedidos de oração
- ✅ Informações de cultos e horários
- ✅ Integração com MongoDB

## 🛠️ Tecnologias

- Node.js + TypeScript
- Express.js
- MongoDB + Mongoose
- Twilio WhatsApp API
- Winston + Morgan (logging)
